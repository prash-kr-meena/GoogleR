           <br> `new_list = old_list.copy()`
        2. You can slice it
           <br> `new_list = old_list[:]`
        3. You can use the built in `list()` function
           <br> `new_list = list(old_list)`
        4. You can do List Comprehension
           <br> `[i for i in old_list]`
